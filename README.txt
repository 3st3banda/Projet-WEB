# Projet-WEB
Wiki Valorant

Présentation du projet
Le projet consiste à réalisé un site web sous forme d’un Wiki, pour un projet universitaire de premier semestre.
La finalité de ce projet est de pouvoir présenter un Wiki sur le thème de Valorant composés d’un maximum d’information est de Tips sur le jeu aux utilisateurs qui s’y intéressent.

Les contraintes potentielles
	-Le manque de temps	
	-Le manque de contenue (car nous ne sommes pas des professionnels sur Valorant)
-Le manque de compétences techniques (Si nous sommes ambitieux sur le rendu visuel final il nous faudra des compétences technique suffisante pour les assumer.)

Prestations Attendues
-J’aimerais que chacun fasse son travail de manière sérieuse afin de pouvoir rendre un projet exemplaire, travaillé et approfondi aux maximum.
