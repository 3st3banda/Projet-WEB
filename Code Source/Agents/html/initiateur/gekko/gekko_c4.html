MORDICUS

Equipez-vous de Mordicus. Tirez pour vous lier mentalement avec Mordicus et la diriger en territoire ennemi. Activez pour bondir vers l'avant et exploser, ce qui retient tous les ennemis dans un petit rayon. Quand Mordicus arrive à expiration, elle redevient un globule dormant. Interagissez avec elle pour récupérer le globule et, après un court délai, pouvoir réutiliser Mordicus. Mordicus peut être récupérée une fois.
