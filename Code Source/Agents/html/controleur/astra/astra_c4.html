FORME ASTRALE/DIVISION COSMIQUE

Activez pour entrer en forme astrale, ce qui vous permet de placer des étoiles en tirant. Vous pouvez ensuite réactiver ces étoiles pour les transformer en impulsion nova, nébuleuse ou puits de gravité.

Lorsque Division cosmique est chargée, utilisez le tir secondaire en forme astrale pour viser, puis tirez pour sélectionner deux emplacements. Une division cosmique infinie connectera les deux points sélectionnés. Division cosmique bloque les balles et étouffe fortement les sons.
